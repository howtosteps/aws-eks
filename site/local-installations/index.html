Local Installations

Install AWS CLI

  • Follow the instructions from this page: Installing or updating the latest version of the AWS CLI
  • Test 
    PS C:\Users\aniru\.aws> aws --version
    aws-cli/2.7.4 Python/3.9.11 Windows/10 exe/AMD64 prompt/off

Install eksctl

  • Follow the installation instructions from here : eksctl installation
  • Test
    PS C:\Users\aniru\.aws> eksctl version  
    0.99.0
